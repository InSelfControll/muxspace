Add or Create a Workspace

2

Launch with Terminal UI

Click "🚀 Launch Workspace" to start the terminal UI with all your tools, browsers, and panes

3

Multi-Project Restoration

Click "⭐ Add to Active Projects" to include it in muxspace restore. All active projects will be restored on startup!

🖥️ Terminal Commands

This GUI is for managing workspaces. To actually use them, run these commands in your terminal:

muxspace start <workspace> - Start a workspace with TUI
muxspace restore - Restore all active projects
muxspace list - List all workspaces
muxspace project <name> - Switch to project
